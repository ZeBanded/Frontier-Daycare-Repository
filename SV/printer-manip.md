# Item Printer RNG Manipulation

This guide shows how to abuse RNG to easily get rare items in Scarlet/Violet's Indigo Disk DLC. Focus will especially be placed on rare items useful for raising Pokemon here, but in general, this is a much more efficient way to farm money and get most items in the game.

[![](https://img.youtube.com/vi/j10Zh7DuCcg/0.jpg)](https://youtu.be/j10Zh7DuCcg)

## Text guide

1. Ensure you have the maxed-out Item Printer that looks like a Master Ball.
2. Select the seed you want from the list below, or from [Anubis's Item Printer Seeds](https://gist.github.com/Lusamine/112d4230919fadd254f0e6dfca850471).
3. Hover over "I want to print something".
4. Go to your Switch's System Settings and change the date and time to the time for that seed, but do not press OK yet.
5. Set a timer for the number of seconds in the target time. *Subtract 1 to account for the fade-to-black.* The Gen 5 one [here](https://not-an-aardvark.github.io/rng-timer/) is recommended - set "Calibration" to -11 and "Target Seconds" to the number of seconds.
6. Press OK on the Switch, *and* start the timer, *at the same time*.
7. Return to the game, where you are still hovering over "I want to print something".
8. Press A the *moment* the timer ends. If you are using the RNG timer linked above, you will hear six clicks, the last one playing when the timer ends, which will help with timing.
9. Perform the print! Most will have you print five items.

If you don't get the correct item, you can use the Adjacent tab in [ItemPrinterDeGacha](https://github.com/kwsch/ItemPrinterDeGacha/releases) (works under Wine on Linux) to see if you were early or late, then adjust and/or try again.

### Getting 2x Item Bonus

Set your system clock to `2044-05-06 15:33:00` and print one item anytime within the next ten seconds.

### Getting Poke Ball Lotto

Set your system clock to `2024-06-12 17:58:00` and print one item anytime within the next ten seconds.

## Items not listed

Generally, if an item is in a shop, it is more efficient to roll and sell Ability Patches, then buy them at that shop.

Ability Capsules and vitamins can be bought at any [Chansey Supply](https://bulbapedia.bulbagarden.net/wiki/Chansey_Supply).

Bottle Caps, Fire/Thunder/Water/Leaf Stones, and Sweet/Tart Apples can be bought at any [Delibird Presents](https://bulbapedia.bulbagarden.net/wiki/Delibird_Presents).

Metal Coats can be bought at the Delibird Presents in Levincia.

Razor Claws can be bought at the Delibird Presents in Mesagoza.
