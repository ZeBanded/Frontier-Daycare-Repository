# Frontier-Daycare-Repository
Repository for the Fontier Daycare, aiming to distribute Pokemon to those who couldn't obtain it themselves
This repository contains guides on various aspects that make it possible to mass distribute Pokemon via Pokemon Home GTS, including GTS guides, catching/breeding guides, evolution item farming, experience farming, etc.
This project is a community charity effort, with no profit motive. All distributed Pokemon should be obtained legitimately and should be volunteer work.
