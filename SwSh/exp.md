# Efficient Level Farming in Sword/Shield

## Poke Job Dateskipping

### Selecting the Poke Job

Talk to Rotomi / the PC in any Pokemon Center.

![](/images/swsh/rotomi.png)

Select "Check Poke Jobs" and choose an Exp. Point job that lets you enter up to 30 Pokemon. Send the Pokemon out for a whole day.

![](/images/swsh/poke-job.png)

![](/images/swsh/poke-job-duration.png)

### Performing the dateskip

Travel to the Wild Area, Isle of Armor, or Crown Tundra, and go to an active raid den.

Ensure you're *not* connected online in Y-Comm, then select "Invite Others".

![](/images/swsh/invite-others.png)

Once you're in the waiting room, go to the Switch Home Menu, then System Settings > System > Date and Time.

If "Synchronize Clock via Internet" is enabled, disable it. Then set the Date and Time forward by one day.

![](/images/switch-date-and-time.png)

Go back to the game and abort the raid. All raids should refresh; this means the dateskip worked.

The Poke Job should now be done, go back to the Pokemon Center and collect it!

![](/images/swsh/poke-job-complete-1.png)

![](/images/swsh/poke-job-complete-2.png)

![](/images/swsh/poke-job-complete-3.png)

Note that Pokemon will not evolve via Poke Jobs; you'll still need to use candies to evolve the Pokemon.
